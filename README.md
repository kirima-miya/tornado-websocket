# 概要

Tornado(Python web framework)を用いた WebSocket 通信

# 実行環境

- Anaconda (Python3.8.5 + ライブラリ群)
- VSCode

# 操作

| 目的         | 方法               |
| :----------- | :----------------- |
| サーバー起動 | `server.py` を実行 |
| サーバー停止 | `Ctrl + C`         |

# フォルダ構成

| フォルダ・ファイル名 | 説明                   |
| :------------------- | :--------------------- |
| static               | CSS, JavaScript を格納 |
| templates            | HTML を格納            |
| `server.py`          | サーバー機能           |
